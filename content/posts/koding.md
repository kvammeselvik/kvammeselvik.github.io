## Innleiing {#innleiing}

Ein av dei kanskje aller største fordelane med å skrive tekstar i Emacs orgmode er at eg kan blande koding og tekst. Sjå til dømes på dette:

```python
for i in range(3):
  print(f"{i}^2 = {i**2}, {i}^3 = {i**3}")
```

```text
0^2 = 0, 0^3 = 0
1^2 = 1, 1^3 = 1
2^2 = 4, 2^3 = 8
```

som gir resultatet

```text
0^2 = 0, 0^3 = 0
1^2 = 1, 1^3 = 1
2^2 = 4, 2^3 = 8
```
